# 📊 Regional Performance Dashboard (Heatmap) | Power BI, API & Excel Integration

## 🚀 Overview
This project presents an **interactive Power BI dashboard** designed for **regional performance analysis**, **KPI monitoring**, and **real-time data integration**.  

The solution combines multiple data sources, including **Excel datasets** and **API connections**, to deliver dynamic and actionable **business intelligence insights**.

---

## 🎯 Business Context
Organizations operating across multiple regions require accurate and real-time visibility into performance metrics.  

This dashboard was developed to:
- Monitor **regional KPIs**
- Identify performance gaps using **heatmap-style visualization**
- Enable **data-driven decision-making**
- Provide **real-time status tracking** using API integration
